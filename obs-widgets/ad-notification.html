Description

An overlay for when Twitch ads are about to start or are currently in progress. Experience a modern, clean design that integrates seamlessly into your OBS setup.

Setup

  1. Open Streamerbot and navigate to Servers/Clients → WebSocket Server. Start the WebSocket server with address 127.0.0.1 and port 8080. Enable Auto Start.
  2. Copy the URL below:
    https://tawmae.github.io/overlays/Ads.html
  3. In OBS, create a new browser source and paste the URL. Set the resolution to match your canvas size.
  4. Enjoy your new ad overlay experience!

Tip: Use a nested scene for multiple OBS scenes to avoid creating multiple browser sources.

URL Parameters

Customize your overlay by appending URL parameters. Start with a ? for the first parameter and use & for additional ones.

https://tawmae.github.io/overlays/Ads.html?sound=true&color=violet&font=lato

Color

Change the color of the title and progress bar. Supports color names and hex codes.

color=blue or color=B81A56

Font Family

Set your preferred font. Use %20 for spaces.

font=Arial, font=Poppins, or font=Comic%20Sans%20MS

Sound

Enable sound when the overlay displays.

sound=true

You can simulate events in Streamerbot by creating a trigger (Twitch → Ads), right-clicking, and selecting Simulate Event.
