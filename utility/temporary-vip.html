Setup

1. Copy the Action Import text from above into your clipboard.

2. In Streamer.bot, click the Import button at the top of the window. Paste the text into the Import String section and press Import at the bottom.

3. Now head to the Commands tab and look for the imported commands. Then rightclick the group and select Group -> Enable All.

And you're done!

Commands

Available time formats:

1y / 2mo / 3w / 4d / 5h / 6min / 7s

!ADDVIP [USERNAME] [TIME]

Adds VIP time to the specified user. If [TIME] has been left blank, it will pick the default time from the settings menu, unless a defaultTime argument has been provided. See the available time formats above.

!REMOVEVIP [USERNAME] [TIME]

Removes VIP time. If no time is given or if the removed time exceeds the time the user had left, it will un-vip them immediately.

!CHECKVIP

Allows users to check when their VIP status expires. If a moderator or permanent VIP uses this, it lists all current temporary VIPs.

!REFRESHVIP

A manual refresh that checks for expired VIPs. If any are expired, theyâ€™ll be removed.

Settings UI

The Settings UI allows for some customization.



General
You can change the general settings here. Default Time Amount and Default Time Unit are just a fallback in case you didn't specify any timespan. You add the timespan in commands directly by input (see Commands above) and for any other trigger, you just add a Set Argument subaction with a defaultTime variable and type the timespan in there.

You can also allow stacking (so two redeems add up instead of just refreshing the time) and limit the slots or exclude a user group from becoming a temporary VIP.

Messages
This tab allows you to customize the chat responses. You can make use of the variables listed below each message.
